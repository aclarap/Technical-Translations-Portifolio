O que são APIs?
Interfaces de Programação de Aplicações (APIs) são estruturas disponibilizadas
em linguagens de programação para permitir que desenvolvedores criem
funcionalidades complexas com maior facilidade. Estas abstraem a complexidade de
códigos mais avançados, oferecendo uma sintaxe mais simples para a utilização.
Como um exemplo real, considere o fornecimento de energia elétrica em sua
residência, como casa, apartamento ou outro tipo de moradia. Para utilizar um
aparelho em uma residência, basta conectá-lo a uma tomada, que este funcionará.
O aparelho não deve ser conectado diretamente na fonte de alimentação - isso
seria ineficiente e, caso não seja um eletricista, difícil e perigoso.
Da mesma forma, caso deseje programar alguns gráficos 3D, é muito mais simples
utilizando uma API escrita em uma linguagem de alto nível, como JavaScript ou
Python, em vez de tentar escrever diretamente um código de baixo nível (como
Linguagem C ou C++), que controla a GPU do computador ou outras funções
gráficas.
